# Data

This project uses **CMS Skilled Nursing Facility (SNF) Cost Reports** from the
Healthcare Cost Report Information System (HCRIS), form **2540-2010**.

Raw CSVs are **not committed to this repo** due to file size. Follow the steps
below to download them into this folder before running the notebooks.

**Provider:** Centers for Medicare & Medicaid Services (CMS)

## Download instructions

### Option 1 — Automated (recommended)

Run this script from the **repo root** to download and extract all seven years
in one go. Each year is a zip file containing multiple flat files; the script
extracts the rollup CSV and renames it to match the expected filename.

```bash
#!/usr/bin/env bash
set -euo pipefail

mkdir -p data
cd data

BASE="https://www.cms.gov/research-statistics-data-and-systems/downloadable-public-use-files/cost-reports/cost-reports-by-fiscal-year-items"

declare -A YEARS=(
  [2015]="snf10-dl-2015"
  [2016]="snf10-dl-2016"
  [2017]="snf10-dl-2017"
  [2018]="snf10-dl-2018"
  [2019]="snf10-dl-2019"
  [2020]="snf10-dl-2020"  # Note: 2020 uses a different path — see manual steps below
  [2021]="snf10-dl-2021"
)

for YEAR in 2015 2016 2017 2018 2019 2021; do
  SLUG="${YEARS[$YEAR]}"
  echo "Downloading $YEAR..."
  curl -L -o "${YEAR}_snf.zip" "${BASE}/${SLUG}"
  unzip -o "${YEAR}_snf.zip" -d "${YEAR}_snf_raw"
  # The rollup CSV inside the zip matches SNF10_*_RPT.csv — rename to convention
  ROLLUP=$(find "${YEAR}_snf_raw" -name "*.csv" | head -1)
  cp "$ROLLUP" "${YEAR}CostReport.csv"
  rm -rf "${YEAR}_snf.zip" "${YEAR}_snf_raw"
  echo "  → Saved ${YEAR}CostReport.csv"
done

echo "Done. All files saved to data/"
```

Save this as `scripts/download_data.sh`, then run:

```bash
chmod +x scripts/download_data.sh
bash scripts/download_data.sh
```

## Notes

- Each zip contains multiple flat files (RPT, NMRC, ALPHNMRC). The notebooks
  use the **rollup CSV** (the single wide-format file), which is the one copied
  above. If your zip contains multiple CSVs, check the CMS documentation to
  identify the correct rollup file for SNF form 2540-2010.
- If a URL returns a 404, visit the CMS fiscal year index directly and locate
  the updated link for that year:
  https://www.cms.gov/data-research/statistics-trends-and-reports/cost-reports/cost-reports-fiscal-year
- `processed_data.csv` is generated automatically by running
  `02_preprocessing.ipynb` after the raw files are in place. Do not commit it
  to git — it is already listed in `.gitignore`.
